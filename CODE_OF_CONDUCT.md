# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/),
version 2.1. The full text of the covenant — including expected standards of behaviour
and the complete list of enforcement guidelines — is available at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Our Standards

Please refer to the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
for the full description of expected and unacceptable behaviour in this community.

## Enforcement

Instances of abusive, harassing, or otherwise unacceptable behaviour may be reported
through GitHub Security Advisories:

https://github.com/Tim81/VellumPDF/security/advisories

All reports are handled **confidentially**. Project maintainers are obligated to respect
the privacy and security of the reporter.
